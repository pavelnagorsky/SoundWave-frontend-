SoundWave music site 2022 (frontend) 

Общая информация: 
  Аудиосервис содержит списки аудиозаписей, доступных для прослушивания и скачивания.
  Все аудиозаписи сгруппированы по категориям: топ по прослушиваниям, топ по кол-ву лайков,
  а так же по списку музыкальных жанров. Присутствует возможность поиска аудиозаписи.
  Для каждой аудиозаписи существует её главная страница с полной информацией и комментариями пользователей.

Функциональность: 
  Гость: 
    Регистрация, авторизация.  
    Просмотор записей, прослушивание не более 10 аудиозаписей.
    Скачивание аудиозаписей.
  Авторизированный пользователь: 
    Неограниченное прослушивание аудиозаписей,
    Возможность ставить лайки и писать/редактировать/удалять комментарии.
    Возможность создавать и редактировать собственные плейлисты.
    Редактировать информацию о своем профиле.
  Администратор:
    Создание/редактирование/удаление аудиозаписей, категорий, чужих комментариев.
    Возможность заблокировать других пользователей или сделать их дать им права администратора.

  Технологии:
    React v17,
    React-Router,
    Redux, Redux Saga,
    React-Bootstrap,
    axios.
